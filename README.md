Para instalar y ejecutar este proyecto localmente, sigue estos pasos:

1. Clona el repositorio

2. Navega al directorio del proyecto
    ```bash
    cd tu-repositorio
    ```
3. Abre `index.html` en tu navegador preferido
